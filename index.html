Raport

Membuat project input output nilai raport sederhana menggunakan C++

project ini adalah sebuah program sederhana yang dibuat untuk mengelola dan mencetak nilai rapor siswa. Dikerjakan saat saya berada di kelas 10, project ini merupakan final project setelah puluhan tugas yang diberikan oleh guru. Tujuan dari project ini adalah untuk memberikan pengalaman praktis dalam pemrograman C++ dengan memanfaatkan berbagai konsep dasar, seperti tipe data, array, dan fungsi. Program ini dimulai dengan registrasi pengguna, di mana siswa diminta untuk memasukkan nama, kelas, dan absen. Setelah itu, mereka dapat mengatur username dan password untuk login ke dalam sistem. Keamanan password diterapkan dengan menyamarkan input menggunakan karakter bintang (*), untuk menjaga privasi. Setelah login berhasil, siswa dapat memasukkan nilai untuk berbagai mata pelajaran. Program ini juga menghitung rata-rata, nilai tertinggi, dan terendah berdasarkan input nilai yang diberikan, serta memberikan status kelulusan berdasarkan kriteria yang telah ditetapkan. Dengan menyusun project ini, saya belajar bagaimana mengelola data secara efisien menggunakan struktur pemrograman, serta memahami pentingnya validasi input dan interaksi pengguna dalam aplikasi berbasis teks.

Cars War Game

Membuat mini games Cars War menggunakan ActionScript 2.0

Membuat project Game Cars War Menggunakan ActionScript 2.0 Pada tahun terakhir di kelas 10, saya mendapatkan kesempatan untuk mendalami ActionScript 2.0 melalui serangkaian project. Setelah menyelesaikan puluhan project yang mengasah keterampilan saya dalam pemrograman, guru memberikan tugas akhir yang menantang: membuat sebuah game bertema perang mobil, yang saya beri nama Cars War. project ini melibatkan penggunaan beberapa frame untuk menciptakan alur permainan yang dinamis dan menarik. Dalam frame pertama, kami menyiapkan animasi loading sederhana dengan kode: stop(); loading._xscale = 0; Frame kedua berisi logika permainan inti, di mana saya mengimplementasikan kontrol mobil dan mekanisme permainan menggunakan objek, variabel, dan fungsi yang saling berinteraksi. Contoh kode yang digunakan untuk menggerakkan mobil termasuk penggunaan deteksi kolisi dan penghitungan skor: if (Key.isDown (87)) { // W key car._y -= (Math.cos (Math.PI / 180 * car._rotation)) * 2; car._x += (Math.sin (Math.PI / 180 * car._rotation)) * 2; } Dalam pengembangan game ini, saya juga menambahkan elemen suara untuk meningkatkan pengalaman bermain. Musik latar dan efek suara tembakan memberikan nuansa yang lebih hidup saat pemain mengemudikan mobilnya dalam pertempuran melawan mobil musuh. Di frame terakhir, saya menyiapkan tampilan hasil akhir dengan kode yang menampilkan apakah pemain menang atau kalah, serta memberikan opsi untuk memulai ulang permainan. Dengan pengalaman ini, saya tidak hanya belajar tentang pemrograman game, tetapi juga pentingnya kreativitas dalam desain dan pengembangan interaktif. project Cars War menjadi bukti kemampuan saya dalam mengaplikasikan teori ActionScript 2.0 ke dalam project praktis yang menyenangkan.

Website Hero ML

Membuat website galeri mobile legend dengan berisi panduan mengenai heronya, dan juga berbagai fitur didalamnya

Proyek ini adalah sebuah website galeri Mobile Legend yang menampilkan informasi lengkap mengenai hero-hero di dalam game. Pada website ini, pengguna dapat melihat detail mengenai hero seperti nama, skin, role, serta kemampuan (abilities) yang ditampilkan dalam bentuk ikon. Ketika ikon kemampuan diklik, latar belakang pentagon akan aktif dan deskripsi mendetail dari kemampuan tersebut akan muncul. Proyek ini menggabungkan beberapa fitur seperti: Scroll Content dengan TranslateX - Elemen hero ditampilkan saat pengguna menggulir ke bawah, memberikan efek transisi yang halus sehingga hero-hero muncul secara bertahap. Popup - Saat tombol arrow ditekan, panel popup muncul dengan hasil match hero serta kemampuan mereka. Active dan Remove Active - Fitur ini digunakan untuk mengatur highlight pada kemampuan yang dipilih, di mana hanya satu kemampuan yang bisa diaktifkan pada satu waktu. Sound - Tombol khusus untuk memutar voice line hero, yang dikontrol dengan tombol play/pause sehingga pengguna bisa mendengarkan voice line hero sesuai pilihan mereka. Proyek ini dikerjakan saat kelas 11, menggabungkan elemen HTML, CSS, dan JavaScript untuk menciptakan pengalaman interaktif. Dengan menambahkan fitur seperti audio, transisi elemen saat scrolling, dan tampilan popup, website ini memberikan pengalaman visual dan interaksi yang mendalam bagi pengguna yang ingin mengetahui lebih banyak tentang hero-hero Mobile Legend.

Fruit Quiz

Membuat game edukasi Fruit Quiz untuk anak anak

Dalam proyek final ini, saya mengembangkan sebuah game kuis buah yang dirancang khusus untuk mengedukasi anak-anak tentang berbagai jenis buah dengan cara yang menyenangkan. Menggunakan ActionScript 3.0 dengan mekanisme dalam game yaitu Drag and Drop, dan Button, proyek ini merupakan hasil dari perjalanan panjang melalui puluhan proyek sebelumnya, yang memberikan saya fondasi yang kuat dalam pemrograman dan desain game. Game ini terdiri dari beberapa scene interaktif yang memudahkan anak-anak belajar sambil bermain. Di awal permainan, terdapat scene loading yang menyajikan animasi yang menarik dan musik latar yang menyenangkan, membantu membangun antisipasi sebelum memasuki menu utama. Menu utama menawarkan dua pilihan permainan: "Berhitung" dan "Tebak Gambar," masing-masing dilengkapi dengan instruksi yang jelas. Pada mode "Berhitung," anak-anak dapat menjawab soal matematika sederhana, sementara di mode "Tebak Gambar," mereka harus mencocokkan gambar buah dengan nama yang benar. Setiap interaksi dilengkapi dengan efek suara yang memberikan umpan balik positif atau negatif, membantu anak-anak memahami jawaban mereka. Notifikasi visual, seperti tanda centang atau silang, menambah elemen edukasi dengan cara yang menyenangkan. Selain itu, timer digunakan untuk memberikan tantangan dan meningkatkan fokus anak-anak selama permainan. Dengan fitur-fitur ini, game kuis buah ini tidak hanya menghibur tetapi juga menjadi alat pembelajaran yang efektif, memberikan pengalaman belajar yang menyenangkan dan menarik bagi anak-anak. Proyek ini merupakan langkah besar dalam pengembangan kemampuan saya dalam menggunakan ActionScript 3.0 dan menciptakan aplikasi yang interaktif serta mendidik.
